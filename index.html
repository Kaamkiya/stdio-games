stdio-games

This is a collection of games, animations, and other cool things.
They are coded in Python, Ruby, C++, and Java.

Installation

You need to have git installed to play these games. Run the following command to clone the repo:

        
git clone https://github.com/Kaamkiya/stdio-games.git

Then cd into the directory:

        
cd stdio-games

Playing

Python
The first thing is to cd into the folder: 
          
cd python

Then use the python or the python3 command to play the game:

          
python <GAME>.py

Make sure to replace <GAME> with one of the actual games in the folder!


Ruby
The first step is to cd into the folder: 
          
cd ruby

Then use the ruby command to play the game:

          
ruby <GAME>.rb

Make sure to replace <GAME> with one of the actual games in the folder!


C++
The first thing is to cd into the folder: 
          
cd cpp

Then use the g++ command to compile the game

          
g++ -o a.out <GAME>.cpp

Make sure to replace <GAME> with one of the actual games in the folder!
Then run the following command to run the compiled code:

          
./a.out

Java
The first thing is to cd into the folder: 
          
cd java

Then use the javac command to compile the game

          
javac <GAME>.java

Make sure to replace <GAME> with one of the actual games in the folder!
Then use the java command to run the compiled code:

          
java <GAME>
